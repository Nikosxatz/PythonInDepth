Μια ερώτηση που απασχολεί όσους ξεκινάνε να διδαχθούν οποιαδήποτε γλώσσα προγραμματισμού: 
Γιατί να μάθω αυτή τη γλώσσα και όχι κάποια άλλη; Η αλήθεια είναι ότι αν θέλει κάποιος να θεωρείται 
προγραμματιστής θα πρέπει να μάθει πολλές γλώσσες. Όλες οι γλώσσες προγραμματισμού έχουν αρκετά 
κοινά μεταξύ τους, όμως η κάθε γλώσσα έχει τα δικά της ιδιαίτερα χαρακτηριστικά και δυνατότητες. 
Από κάθε γλώσσα ο επίδοξος προγραμματιστής θα μάθει πράγματα τα οποία θα τον βοηθήσουν να εμπλουτίσει 
τις γνώσεις του και τις δεξιότητές του. Αργότερα, ανάλογα με τον τομέα στον οποίο θα ασχοληθεί, 
θα επικεντρωθεί σε συγκεκριμένες γλώσσες. Άλλες γλώσσες χρησιμοποιούνται για τη δημιουργία δυναμικών 
ιστοσελίδων (Web Programming), άλλες για τη διαχείριση βάσεων δεδομένων (Database Programming) και 
άλλες για την κατασκευή λογισμικού παιχνιδιών (Game Programming). Η Python είναι μια από τις πιο 
σύγχρονες γλώσσες προγραμματισμού και σίγουρα υπάρχουν αρκετοί λόγοι για να ασχοληθεί κάποιος μαζί της:

- Η Python  είναι μια γλώσσα που χρησιμοποιείται ευρέως σήμερα, ειδικά στον τομέα του Web programming.
- Είναι απλή στη χρήση, εύκολη στην εκμάθηση και ο κώδικάς της ιδιαίτερα ευανάγνωστος.
- H Python διαθέτει βιβλιοθήκες, σχεδόν για τα πάντα, γεγονός που επιτρέπει την κατάστρωση απαιτητικών 
εφαρμογών με ελάχιστο κώδικα.
- Είναι μια  γλώσσα δυναμικού τύπου με αυτόματη διαχείριση μνήμης, πράγμα που μας γλυτώνει από πολλά προβλήματα, 
ιδιαίτερα τους αρχάριους στον προγραμματισμό.
- Είναι μια γλώσσα που υποστηρίζει και τον αντικειμενοστρεφή προγραμματισμό πράγμα που σημαίνει ότι σταδιακά 
μπορούμε να μπούμε στη φιλοσοφία του.
- Χρησιμοποιείται επίσης στον προγραμματισμό ρομποτικών διατάξεων. Ιδιαίτερα, ο μικροϋπολογιστής 
Raspbery Pi, ο οποίος διατίθεται με εγκατεστημένο το περιβάλλον της Python, αποτελεί τη βάση για την 
κατασκευή ποικίλων ρομποτικών εφαρμογών. 
- Είναι της μόδας!
